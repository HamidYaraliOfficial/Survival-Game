# English

## Overview
**Survival Game** is a modern 2D action-survival platform game developed with Python and PyQt6. The game features multiple levels, enemies, bosses, particle effects, character progression, experience and leveling systems, grenades, special abilities, multilingual support, and light/dark themes.

## Features

- Modern 2D platform gameplay
- Multiple enemy types
- Boss battles
- Experience and level-up system
- Combo scoring system
- Grenades and special attacks
- Particle effects and animations
- Multilingual interface
  - English
  - Persian (فارسی)
  - Chinese (中文)
- Dark and Light themes
- Responsive user interface
- Keyboard controls

## Requirements

- Python 3.10 or newer
- PyQt6

## Installation

### Install Required Libraries

```bash
pip install PyQt6
```

### Run the Game

```bash
python Survival.py
```

## Controls

| Key | Action |
|------|---------|
| A / D | Move |
| W / Space | Jump |
| S | Duck |
| Shift | Run |
| J | Shoot |
| K | Throw Grenade |
| L | Special Ability |
| P | Pause / Resume |

## Game Systems

### Character Progression
- Gain XP from defeating enemies.
- Level up to improve health and abilities.
- Unlock stronger performance through progression.

### Enemies
- Soldier
- Scout
- Heavy
- Drone
- Boss

### Combat
- Standard shooting
- Spread shooting
- Grenades
- Special attacks
- Combo bonuses
